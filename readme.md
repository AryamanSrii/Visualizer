

# Sorting Algorithms Visualizer

Sorting Algorithms Visualizer (SAV) produes visuals instead of long codes to understand the algorithms.

- Merge Sort
- Bubble Sorting
- Selection Sort
- Insertion Sort
- Quick Sort

## Live at 

https://visualizer.tk/
